# Task Manager

## Getting Started

### Prerequisites

- Node.js 20.x
- pnpm

## Api

1. Install dependencies
   ```sh
   pnpm install
   ```
2. Migrate database and seed data
   ```sh
   pnpm db:migrate
   ```
3. Start the server
   ```sh
   pnpm dev
   ```
4. Open the browser and navigate to http://localhost:3000/api

## Frontend

1. Install dependencies
   ```sh
   pnpm install
   ```
2. Start the server
   ```sh
   pnpm dev
   ```
3. Open the browser and navigate to http://localhost:5173
